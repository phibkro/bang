## The gate (step 3) — real exit code

```bash
nix develop --command just verify ; echo "EXIT=$?"     # selfcheck + lake build + audit
```

NEVER `| head`/`| grep` the result and trust the exit — that is *the* false-green
in this repo (memory: `nix-build-verify-exit-codes`). The pre-commit hook already
runs `just verify` and blocks theorem-statement changes unless
`STATEMENT_CHANGE_OK=…` justifies them.

Repo-specific invariant guards (the `tools/audit.sh` rung):
- `sorry` only in theorem bodies, never in definitions / `Bang/Audit.lean`.
- No new `axiom` declarations; per-theorem `#print axioms ⊆ {propext,
  Classical.choice, Quot.sound}` (+ `sorryAx` while a body is `sorry`).
- Five kernel primitives only (thunk · force · effect rows · handlers · STM).
- Diff-scoped fast loop: `nix develop --command just check FILE` (G0/G1).

## Artifact map (step 4) — by volatility

| artifact | Diátaxis mode | volatility | when to survey |
|---|---|---|---|
| `CONTEXT.md` | volatile-state | ALWAYS | every session/checkpoint |
| `CLAUDE.md` | reference + how-to (invariants, glossary, verify-cmd) | HIGH (always-loaded) | any kernel-rule, command, or invariant change |
| `paths/PATH-*.md` | volatile-state | ALWAYS (while a path is in flight) | every in-path checkpoint |
| `ROADMAP.md` | explanation | LOW | a checkpoint (◊) moved |
| `docs/decisions/*` (ADRs) + `README` index | explanation | LOW / high-stakes | a design decision was made |
| `docs/notes/OPEN_QUESTIONS.md` | explanation | MED | a Q opened/resolved |
| `ONBOARDING.md`, `README.md` | tutorial / how-to | LOW | setup or orientation changed |
| `references/README.md` + `refs.bib` | reference | MED | a paper added or a citation changed |
| `docs/notes/spec-proof-discipline.md` | how-to | MED | PROOF_ORDER or proof conventions changed |

Note: this repo has **no generated reference** yet — every Reference doc is
hand-maintained, so each is in the survey when its surface moves. A `#print
axioms`-derived burndown (`tools/burndown.sh`) is the one *generated* artifact;
it cannot drift.
